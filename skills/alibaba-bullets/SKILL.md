# Alibaba Bullets

## 适用场景
用于生成阿里国际站五点卖点。卖点必须服务 B2B 采购决策，不写偏 C 端的零售情绪文案。

## 输入字段
- 产品名称
- 产品中心词
- 核心关键词
- 属性词
- 应用场景
- 材质
- 规格
- 功能
- 定制能力
- MOQ
- 样品
- 交期
- 包装
- 认证
- 目标国家
- 目标买家
- 补充说明

## 执行步骤
1. 第一点写采购适配：说明适合哪些目标买家和使用场景。
2. 第二点写产品价值：使用材质、规格、功能等可验证信息。
3. 第三点写定制能力：说明 logo、颜色、尺寸、包装、OEM/ODM 等能力。
4. 第四点写订单执行：说明 MOQ、样品、交期、包装。
5. 第五点写市场信任：说明认证、目标市场适配或质检沟通。
6. 每一点都要解释买家收益，不只罗列参数。

## 输出格式
```text
1. Procurement Fit: ...
2. Verified Product Value: ...
3. Customization Support: ...
4. Order Execution: ...
5. Market Confidence: ...
```

## 禁止事项
- 禁止简单照搬亚马逊五点。
- 禁止只写 “high quality / hot sale / popular” 这类空泛表达。
- 禁止编造 MOQ、交期、认证、付款、售后。
- 禁止承诺未提供的库存、价格、质保、客户案例。

## 示例
输入：
```text
目标买家：retailers, distributors
材质：304 stainless steel
规格：500ml / 750ml / 1000ml
定制能力：logo printing, custom color
MOQ：500 pcs
样品：3-5 days
交期：15-25 days
认证：LFGB, FDA
```

输出：
```text
1. Procurement Fit: suitable for retailers and distributors sourcing reusable drinkware.
2. Verified Product Value: 304 stainless steel and multiple capacity options help buyers compare product lines.
3. Customization Support: logo printing and custom colors support private-label projects.
4. Order Execution: MOQ 500 pcs, sample in 3-5 days, and 15-25 days lead time support planned purchasing.
5. Market Confidence: LFGB and FDA information can support compliance review before bulk order.
```
