# Bullet Skill

## 适用场景
用于生成阿里国际站商品五点卖点。卖点必须服务 B2B 采购决策，而不是简单照搬亚马逊零售五点。

## 输入字段
- 产品名称：{{productName}}
- 产品中心词：{{productCore}}
- 核心关键词：{{coreKeyword}}
- 品牌/营销词：{{brandMarketing}}
- 属性词：{{attributeWords}}
- 应用场景：{{scenarios}}
- 材质：{{material}}
- 规格：{{specs}}
- 功能：{{functions}}
- 定制能力：{{customization}}
- MOQ：{{moq}}
- 样品：{{sample}}
- 交期：{{leadTime}}
- 包装：{{packaging}}
- 认证：{{certifications}}
- 目标国家：{{targetCountries}}
- 目标买家：{{targetBuyers}}
- 补充说明：{{notes}}

## 执行步骤
1. 第一条说明产品定位和采购适配场景。
2. 第二条说明材质、规格、功能等可验证的产品价值。
3. 第三条说明定制能力，包括 logo、颜色、尺寸、包装或 OEM/ODM。
4. 第四条说明采购执行信息，包括 MOQ、样品、交期和包装。
5. 第五条说明信任背书，包括认证、目标市场适配、售后沟通或质检流程。
6. 每条都要体现买家收益，避免只列参数。

## 输出格式
```text
1. [采购价值标题]: [面向 B2B 买家的说明]
2. [采购价值标题]: [面向 B2B 买家的说明]
3. [采购价值标题]: [面向 B2B 买家的说明]
4. [采购价值标题]: [面向 B2B 买家的说明]
5. [采购价值标题]: [面向 B2B 买家的说明]
```

## 禁止事项
- 禁止使用偏 C 端的情绪化零售话术。
- 禁止只重复参数，不解释采购价值。
- 禁止承诺未提供的库存、价格、认证、交期或售后政策。
- 禁止夸大性能或使用无法验证的绝对化表达。
