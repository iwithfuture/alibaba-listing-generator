# Final Output Skill

## 适用场景
用于把标题、五点卖点、FAQ、详情页结构和质检结果整理成运营可复制、可归档、可交付的最终商品发布文档。

## 输入字段
- 产品 brief：{{productBrief}}
- 标题结果：{{titleOutput}}
- 五点卖点：{{bulletsOutput}}
- FAQ：{{faqOutput}}
- 详情页结构：{{detailOutput}}
- 质检结果：{{qcOutput}}

## 执行步骤
1. 按商品发布顺序整理输出内容。
2. 保留标题、五点卖点、FAQ、详情页结构和质检结果的模块边界。
3. 对质检中标记为“需修改”的内容做醒目标注。
4. 保持格式简洁，方便复制到阿里国际站后台、文档或表格。
5. 不改变已生成内容的含义，只做结构化整理。

## 输出格式
```text
# 阿里国际站商品发布内容

## 标题

## 五点卖点

## FAQ

## 详情页结构

## 质检结果

## 待补充信息
```

## 禁止事项
- 禁止在最终输出中混入 prompt 原文。
- 禁止删除质检模块提出的风险和待补充项。
- 禁止重写为夸张营销文案。
- 禁止添加输入资料中不存在的具体承诺。
